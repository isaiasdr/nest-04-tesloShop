# Teslo API

1. Clone the proyect
2. Install the dependecies
```
yarn
```

3. Copy the file __.env.template__ and rename to __.env__

4. Update the env variables

5. DB up
```
docker-compose up -d
```

6. Run the app:
```
yarn start:dev
```

7. Execute Seed
```
http://localhost:3000/api/seed
```
